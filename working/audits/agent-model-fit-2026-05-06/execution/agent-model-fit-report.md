# Agent Model-Fit Audit — 2026-05-06

**Scope:** All 27 agent prompts in `/Users/mnoth/source/asoiaf-chat/.claude/agents/*.md`.

**Methodology:** Each agent prompt was read in full. Recommendation reflects what the prompt actually demands — token volume, reasoning chain length, ambiguity-handling load, output-structure complexity — not the agent's blurb. Stub agents (prompt body is "TODO" / "STATUS: Stub") are flagged `STUB` rather than rated; they need re-audit when the body lands. Recommendations apply Matt's standing model-fit policy (default cheapest viable; Opus only when reasoning depth genuinely requires it; smoke-test before upgrading).

## Closing summary

Of the 27 agents audited: **6 recommended for downgrade** (mechanical-extractor, wiki-ingester, citation-validator, orphan-edge-finder, prose-edge-classifier, schema-drift-auditor — all Opus → Sonnet, with prose-edge-classifier smoke-test gated); **2 additional downgrades to Haiku** (status-reporter from default-Opus, duplicate-detector from Opus); **4 already at correct cheapest tier** (script-builder Sonnet, prose-edge-reviewer Sonnet, fleet-stats-reviewer Sonnet, plus an implicit "stay" on the runnable Sonnet agents); **2 genuinely need Opus** (cross-identity-detector, cross-identity-reviewer — high-stakes, low-volume `SAME_AS` decisions); **13 are STUBs** flagged for re-audit when their prompt bodies are written. The biggest expected savings are in Stage 1 audits and (smoke-test gated) Stage 2a prose-edge classification.
