# Taxonomy Candidates

> **Purpose:** Entities encountered during wiki scrape that don't fit the current 6 node types (Characters, Locations, Factions, Artifacts, Prophecies, Events). Populated by the wiki scraper's category discovery mode.
>
> **Status:** Awaiting wiki scrape. AWOIAF is behind Cloudflare — needs browser cookies to access. See `scripts/wiki-scraper.py --help` for setup instructions.

---

## Current Node Types (from architecture.md)

| Type | Description |
|------|-------------|
| character | Named individual |
| location | Named place |
| faction | Organization or alliance |
| artifact | Object of narrative significance |
| prophecy | Prophetic statement or vision |
| event | Confirmed plot event |
| theory | Interpretive framework |
