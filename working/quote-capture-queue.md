# Quote Capture Queue

> **Purpose:** Lightweight backlog for the firm "capture load-bearing quotes" rule (`feedback_capture_quotes_during_research`). When a session is in the chapter/wiki text for any reason and finds a quotable line (death, feast/food, hospitality, vivid description, key dialogue, prophecy) but doing the *full* graph-attach right now would derail the task or bloat context, drop a cheap pointer here instead. A later focused pass does the real attach (edge `evidence_quote`/`evidence_ref` or node `## Quotes`).
>
> **The rule is "never lose the line," not "always write inline."** A logged `file:line` + note fully satisfies it.

## Format

One row per quote. Keep it to a single line — this is a pointer, not the work.

| file:line | target (edge or node, or TBD) | ≤1-sentence note |
