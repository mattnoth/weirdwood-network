Featured quotes[ ]

Featured quotes are quotes from the A Song of Ice and Fire universe that we find funny, insightful, witty, etc. The goal is to provide for a different quote to be displayed on the Main Page each day. So don't be shy, share with us your favorites!

Adding new quotes[ ]

To add a new quote:

Make a new subpage. The name must be the next available number. For example, if the last quote in the list is 15, create a new subpage by the name Feature quote/16.

Write down your quote and include following code at the bottom: <noinclude>[[Category:Feature quotes]]<references /></noinclude>.

Add it to the list below List of Quotes (again with the next available number) and change the "total quotes=73" number to the new total.
